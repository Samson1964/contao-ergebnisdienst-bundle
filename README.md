# DSB-Ergebnisdienst

**Frank Hoppe**

Mit dieser Erweiterung kann die Schnittstelle des Ergebnisdienstes (Bundesligen und DSOL) des Deutschen Schachbundes abgefragt werden. Für den Zugriff auf die Schnittstelle ist ein Schlüssel erforderlich, die beim Programmierer Matthias Berndt (<matthias.berndt@web.de>) angefordert werden kann. 

Wer die Erweiterung nicht nutzen möchte und stattdessen etwas Eigenes programmiert, kann die Schnittstellenbeschreibung unter <https://ergebnisdienst.schachbund.de/json/> und <https://dsol.schachbund.de/json/> nutzen. 

## Installation und Anwendung

Unter System -> Einstellungen muß der oder die Schlüssel eingetragen werden. Danach steht das Inhaltselement "DSB-Ergebnisdienst" zur Verfügung und kann genutzt werden.
